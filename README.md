# Len's Slice

This project is a small Python script built as part of the Codecademy Data Engineer Career Path. It practices fundamental Python data structures, focusing specifically on list methods, sorting, and slicing techniques to organize business data.

## Project Objectives
- Manage an inventory of pizza toppings and their respective prices.
- Organize data into two-dimensional lists to pair prices with toppings.
- Sort and slice datasets to extract key business insights (e.g., cheapest options, priciest items, popular selections).

## Features
- **Data Pairing:** Dynamically combines price and topping datasets into a unified 2D list.
- **Inventory Metrics:** Calculates the total number of pizza varieties sold.
- **Sorting & Slicing:** Sorts inventory automatically from lowest to highest price and handles custom customer requests by slicing specific ranges of the dataset.

## Technologies Used
- **Python 3**

## How It Works
The script initializes lists for pizza prices and toppings, converts them into a combined 2D list using iterative pricing pairs, and applies built-in methods like `.sort()`, `.pop()`, and `.insert()` to modify the menu as inventory changes.
